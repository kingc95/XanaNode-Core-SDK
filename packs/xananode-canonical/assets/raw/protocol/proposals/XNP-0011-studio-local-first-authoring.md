# XNP-0011: Studio Local-First Authoring

Status: Draft

## Summary

Captures the Studio product direction as local-first Git-backed authoring for knowledge substrates.

## Decision

Studio should treat a substrate as a local Git repository and expose user-facing concepts before raw Git mechanics.

Author-facing language:

- save snapshot
- draft path
- bring changes together
- propose changes
- what changed
- history

The Studio MVP should focus on opening substrate repos, editing nodes, adding relationships, validating, previewing with the Hugo implementation, saving snapshots, and publishing.
