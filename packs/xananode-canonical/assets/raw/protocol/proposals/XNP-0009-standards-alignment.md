# XNP-0009: Standards Alignment Profiles

Status: Accepted

## Summary

Defines optional interoperability profiles for JSON-LD, Web Annotation selectors, PROV-O, content IDs, RO-Crate, and nanopublication-style claim packages.

## Decision

XanaNode keeps its native artifacts simple, but provides mappings and companion profiles for standards-aware tools.

Accepted profiles:

- JSON-LD context
- Web Annotation selector names
- PROV-O provenance mapping
- optional content IDs and version IDs
- RO-Crate-compatible preservation metadata
- nanopublication-like claim packaging

These profiles are optional for basic substrates but recommended for archival, scholarly, legal, scientific, and federated deployments.
