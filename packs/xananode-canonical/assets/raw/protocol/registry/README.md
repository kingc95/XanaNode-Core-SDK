# Registry

The registry directory lists known namespaces, schemas, implementations, and related resources.

This is not a central authority over truth.

It is a discovery aid.

Known implementations are listed in [known-implementations.json](known-implementations.json). Implementation repositories should point back to the protocol version and schema artifacts they target.

Known federation targets are listed in [federation-targets.json](federation-targets.json). These are discoverable online substrate sources that Workspace tools can clone, identify by branch and commit, and mount into another substrate without absorbing their records.
